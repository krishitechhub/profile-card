#  Modern Profile Card

This is my first frontend web development project built using **HTML** and **CSS**.

The project is a modern profile card that helped me practice HTML structure, CSS styling, Flexbox, gradients, and responsive design.

##  Features

- Modern UI Design
- Responsive Layout
- Gradient Background
- Circular Profile Picture
- Hover Effects
- Skills Section
- Statistics Section
- Action Buttons

## Technologies Used

- HTML5
- CSS3
- Flexbox
